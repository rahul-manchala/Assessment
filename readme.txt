/*********** Steps To Run App ***********/
Without Local server
1. Close chrome app and all its instances from process.
2. Launch chrome app by disabling web security ("system path for chrome.exe" --allow-file-access-from-files --disable-web-security)
3. Drag and drop index.html to the chrome browser.

With local server (grunt task)
1.From the project location in cmd run the following commands. (Note: you should install node before)
2. npm install
3. grunt
